```

```
## kalkulator
Simple scientific calculator in java with only 4 operators ( '\+' , '\-' , '\*' , '\/' ) with brackets ( '(' , ')' )
### Format Input:
Read input of string to calculate line by line
#### Example:
```
(2+3)
2*(3-4)
2.44*(64+6*6)
```
